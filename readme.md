## 개발 범위

* 플레이어블 캐릭터
	- 원거리 캐릭터 1명, 근거리 캐릭터 1명
	- 스킬은 각자 3개로 구현
* 몬스터 3종
	- 3~4가지 정도의 다양한 패턴
* 스테이지 3종
	- 각자 다른 배경의 스테이지
	- 장애물 등을 배치해서 몬스터의 패턴과 상호작용 가능하게 구현
